# Angular [![My Skills](https://skillicons.dev/icons?i=angular)](https://skillicons.dev)

## Descripción

- Repositorio principal para el seguimiento del primer curso de Angular.
- En este repositorio se mantendrán las notas y enlaces; cada proyecto práctico se alojará en su propio repositorio.

## Estructura recomendada

- /notes              -> Notas generales del curso (aquí se guardan archivos como instalacion.md)
- /DevTalles/ejercicios -> Ejercicios prácticos del curso
- /DevTalles/documentos -> Guías y documentos relacionados
- Otros repositorios para proyectos completos del curso

## Uso

- Guardar en /notes las anotaciones que se vayan tomando.
- Colocar ejercicios y guías dentro de DevTalles en las subcarpetas indicadas.
- Para nuevos cursos, crear una entrada en este README y un repositorio separado para el curso correspondiente.
